# 회원가입, 로그인 구현
간단한 로그인, 회원가입 페이지를 구현한 프로젝트입니다.

## 구현 목표
### 개발 도구 사용
- parcel
- handlebars
- REST client
- babel
- typescript
### 기능 구현
- 회원가입 기능
- 로그인 기능
- url 해시를 이용한 라우팅
- 주소 검색 api 사용
- 간단한 node 서버 직접 구현

## 실행
### 클라이언트
```
npm run start:client
```
### 서버
```
npm run start:server
```
